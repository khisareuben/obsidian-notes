
## 🛡️ Cybersecurity Roadmap (Company-Oriented)

### **Stage 1: Foundations (Entry-Level Roles: SOC Analyst, IT Support)**

- **What they do in companies**: Monitor systems, detect suspicious activity, escalate incidents.
    
- **Skills to learn**:
    
    - Networking basics (TCP/IP, DNS, HTTP, VPNs)
        
    - Operating systems (Windows/Linux security basics)
        
    - Security fundamentals (firewalls, antivirus, IDS/IPS)
        
- **Practical resources**:
    
    - CompTIA Security+ syllabus (even if you don’t take the exam, it’s a structured roadmap)
        
    - Hands-on labs: TryHackMe beginner paths, HackTheBox Academy
        

### **Stage 2: Application Security (Relevant to Mobile Dev)**

- **What they do in companies**: Ensure apps are secure, test for vulnerabilities, enforce secure coding practices.
    
- **Skills to learn**:
    
    - OWASP Top 10 (SQL injection, XSS, insecure authentication, etc.)
        
    - Secure coding in Kotlin/Java (input validation, encryption, secure APIs)
        
    - Mobile app security testing (reverse engineering, API security)
        
- **Practical resources**:
    
    - OWASP Mobile Security Testing Guide
        
    - Labs: Juice Shop (OWASP vulnerable app), Mobile-specific labs on TryHackMe
        

### **Stage 3: Intermediate Roles (Incident Response, Penetration Testing)**

- **What they do in companies**: Investigate breaches, simulate attacks, report vulnerabilities.
    
- **Skills to learn**:
    
    - Digital forensics basics (logs, evidence collection)
        
    - Penetration testing tools (Metasploit, Burp Suite, Wireshark)
        
    - Threat intelligence (how attackers operate)
        
- **Practical resources**:
    
    - MITRE ATT&CK framework (industry standard for attack techniques)
        
    - Labs: HackTheBox penetration testing paths
        

### **Stage 4: Advanced Roles (Cybersecurity Engineer, Architect)**

- **What they do in companies**: Design secure systems, manage enterprise security, oversee compliance.
    
- **Skills to learn**:
    
    - Cloud security (AWS, Azure, GCP)
        
    - Identity & access management (IAM, MFA, SSO)
        
    - Governance, Risk & Compliance (ISO 27001, GDPR, PCI DSS)
        
- **Practical resources**:
    
    - Cloud provider free training (AWS Security Essentials, Azure Security Fundamentals)
        
    - CISSP syllabus for broad coverage
        

## ⚖️ How This Fits With Mobile Dev

- **Direct synergy**: Application security overlaps perfectly with mobile dev. You’ll learn to build apps that are secure from the ground up.
    
- **Career flexibility**: You can start as a mobile dev, then specialize in mobile app security—a niche that’s rare and valuable in Kenya’s fintech ecosystem.
    
- **Future-proofing**: Adding cybersecurity makes you versatile: you can work in dev teams, security teams, or freelance globally.
