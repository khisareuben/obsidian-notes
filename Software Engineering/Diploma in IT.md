
## Diploma Options Compared

| Diploma                              | Focus                                         | Typical Careers                            | Fit for Your Goals                                                             |
| ------------------------------------ | --------------------------------------------- | ------------------------------------------ | ------------------------------------------------------------------------------ |
| **Software Design & Programming**    | Building apps, coding, software architecture  | Mobile dev, web dev, software engineer     | ✅ Strong match — aligns with your Android/Kotlin path                          |
| **Network & Systems Administration** | Managing servers, networks, IT infrastructure | System admin, IT support, network engineer | ⚖️ Good if you want stable IT jobs, but less about app dev                     |
| **Computer Systems**                 | Broad mix of hardware + software basics       | IT technician, systems analyst             | ⚖️ Generalist — useful, but not specialized                                    |
| **Computer Science**                 | Theory + programming + algorithms             | Software engineer, researcher, dev roles   | ✅ Strong academic foundation, versatile for dev + finance tech                 |
| **Computer Studies**                 | Introductory, broad overview                  | Entry-level IT, office tech                | ❌ Too basic if you want advanced dev skills                                    |
| **Computer Programming**             | Focused coding skills                         | Programmer, junior developer               | ✅ Practical, but narrower than Software Design & Programming                   |
| **Certified ICT**                    | Broad ICT skills, often exam-based            | ICT officer, IT support                    | ⚖️ Good for government/NGO jobs, less dev-focused                              |
| **Cyber Security**                   | Protecting systems, ethical hacking           | Security analyst, penetration tester       | ⚖️ Growing field, but not directly tied to app dev unless you specialize later |
