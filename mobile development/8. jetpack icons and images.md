```kotlin
IconButton(onClick = {}) {
```

```kotlin
painter = painterResource(id = R.drawable.imagename)
```


### CircularProgressIndicator

This is the circular round moving bar/icon e.g. when loading online content

```kotlin
CircularProgressIndicator(
	modifier = Modifier
		.height(16.dp)
		.width(16.dp)
	strokewidth = 2.dp
	color = Color.Cyan
)

```
